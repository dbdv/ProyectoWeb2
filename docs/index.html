Sobre la web

Animal Curiosity es una web dedicada a, como indica el nombre, curiosidades sobre el mundo animal. Acá podrán encontrar datos, imágenes y tips sobre sus animales favoritos, además de poder conocer nuevos animales.

Si te gusta lo que esta web tiene para ofrecer, compártela con quienes creas que les podría interesar.

Además si quieres colaborar con el proyecto, te dejo el enlace a mi cafecito.
